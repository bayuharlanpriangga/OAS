Groq API — 100% Gratis, Selamanya!
AI Orias pakai Llama 3.3 70B via Groq. Tidak perlu kartu kredit, tidak ada biaya apapun.

Langkah-langkah (±2 menit):

1. Buka console.groq.com/keys di tab baru
   ↳ Daftar / login gratis (bisa pakai akun Google)

2. Klik tombol Create API Key

3. Isi form yang muncul:
   • Display Name → isi bebas, contoh: Orias Key 1
   • Expiration → pilih No expiration (biar tidak perlu diperbarui)
   • Klik Submit

4. Key langsung muncul — bentuknya gsk_xxx...
   ↳ Copy sekarang! Key hanya tampil SEKALI, setelah ditutup tidak bisa dilihat lagi

5. Paste di kotak bawah → klik + Tambah → selesai!

Buat 3-5 key dari akun yang sama = limit 90-150 req/menit.
Key tersimpan di browser — tidak perlu setup ulang saat buka web lagi.
Key hanya tersimpan di browser kamu — tidak dikirim ke server mana pun selain langsung ke Groq.
