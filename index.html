PPh Pasal 21 — Pegawai Tetap
📋 PMK 168 Tahun 2023
📆 Rekonsiliasi Akhir Tahun (Desember)
💡 Kosongkan = asumsi gaji sama tiap bulan (otomatis × 11)
💡 Total semua bonus yang diterima sepanjang tahun
⚠️ PMK 168/2023: Upah ≤ Rp450rb/hari tidak dipotong (TER Harian). Rp450rb–Rp2,5jt: TER Harian 5% dari selisih. Di atas Rp2,5jt/hari: Ph.Bruto × 50% × Tarif Ps.17.
🏛️ PMK 168/2023: Komisaris/Dewan Pengawas yang menerima penghasilan tidak teratur (honor, tantiem, dll) — PPh 21 = Ph.Bruto × TER Bulanan, diterapkan per masa pajak.
💼 Tidak Berkesinambungan: Honor diterima sekali/tidak rutin. PPh 21 = Ph.Bruto × 50% × Tarif Ps.17 — dihitung per pembayaran, tanpa akumulasi.
📊 Berkesinambungan: Honor diterima rutin/berulang dari satu pemberi kerja. PPh 21 dihitung secara akumulatif — isi kumulatif bruto sebelum masa ini agar tarif progresif tepat.
🎯 PMK 168/2023: Peserta kegiatan (peserta lomba, rapat, konferensi, dll) yang menerima hadiah/imbalan — PPh 21 = Ph.Bruto × Tarif Ps.17. Per masa pajak atau saat terutang.
🏦 PMK 168/2023: Peserta program pensiun (pegawai) yang melakukan penarikan dana pensiun di awal — PPh 21 = Ph.Bruto × Tarif Ps.17. Diterapkan per masa pajak.
📋 PMK 168/2023: Mantan pegawai yang menerima jasa produksi, tantiem, gratifikasi, bonus, atau imbalan lain — PPh 21 = Ph.Bruto × Tarif Ps.17. Per masa pajak.
🏛 PMK 168/2023: PNS/TNI/POLRI/Pejabat Negara & Pensiunan — penghasilan tetap teratur dipotong dengan TER Bulanan tiap masa (kecuali masa pajak terakhir Desember). Termasuk gaji ke-13 & gaji ke-14 yang dibayar pada masa yang bersangkutan.

🤖 Setup Orias Assisten (Gratis)
Belum ada key
🆓 Groq API — 100% Gratis, Selamanya!
AI Orias pakai Llama 3.3 70B via Groq. Tidak perlu kartu kredit, tidak ada biaya apapun.

📋 Langkah-langkah (±2 menit):

1. Buka console.groq.com/keys di tab baru
   ↳ Daftar / login gratis (bisa pakai akun Google)

2. Klik tombol Create API Key

3. Isi form yang muncul:
   • Display Name → isi bebas, contoh: Orias Key 1
   • Expiration → pilih No expiration (biar tidak perlu diperbarui)
   • Klik Submit

4. Key langsung muncul — bentuknya gsk_xxx...
   ↳ ⚠️ Copy sekarang! Key hanya tampil SEKALI, setelah ditutup tidak bisa dilihat lagi

5. Paste di kotak bawah → klik + Tambah → selesai! ✅

💡 Buat 3-5 key dari akun yang sama = limit 90-150 req/menit.
Key tersimpan di browser — tidak perlu setup ulang saat buka web lagi.
🔒 Key hanya tersimpan di browser kamu — tidak dikirim ke server mana pun selain langsung ke Groq.
